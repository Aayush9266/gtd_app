## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/gtd-app.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the app**
   ```bash
   expo start
   ```

Scan the QR code with the [Expo Go app](https://expo.io/client) or run on an emulator.
